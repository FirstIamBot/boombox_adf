Custom Web UI

This is a placeholder. To use the full Web UI:

# Copy and build frontend
cp -r ../../../frontend ./frontend
cd frontend && npm install && npm run build && cd ..

# Copy built files
cp frontend/dist/index.html www/
mkdir -p www/assets
cp frontend/dist/assets/*.gz www/assets/

# Rebuild and flash
idf.py build flash

Or create your own frontend using the REST API.
